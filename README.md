# Planetary Motion Simulator

A Python simulation that models planetary motion using Newton's law of gravitation.

## Overview

This project simulates the motion of celestial bodies under gravitational attraction. The simulator demonstrates orbital mechanics and planetary trajectories based on Newtonian physics.

## Features

- Planetary orbit simulation
- Gravitational force calculations
- Real-time trajectory visualization
- Adjustable physical parameters

## Physics Concepts

- Newton's Law of Gravitation
- Orbital Mechanics
- Circular and Elliptical Orbits
- Conservation of Energy
- Conservation of Momentum(Approximately accurate, slight simulation errors might appear due to Euler numerical method)

## Technologies Used

- Python
- NumPy
- Matplotlib

## Numerical Concepts

- Euler numerical method
